# Domora 2.0 — Gioco HTML

Versione web del gioco Domora 2.0, esportata da Godot.

## Avvio locale

Apri `domora.html` in un browser moderno oppure servi la cartella con un server HTTP locale.

## File principali

- `domora.html` — pagina principale del gioco
- `domora.js` — runtime del gioco
- `domora.audio.worklet.js` / `domora.audio.position.worklet.js` — audio worklets
- `_headers` — header HTTP (es. per Cloudflare Pages)
